## ğŸš€ Getting Started

### Prerequisites

- Node.js v16 or higher
- npm or yarn package manager

### Installation

```bash
# Clone the repository
git clone https://github.com/yashsoni27/f1-dash.git

# Install dependencies
cd f1-dash
npm install --legacy-peer-deps

# Start the development server
npm run dev
```
